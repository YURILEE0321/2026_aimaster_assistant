1. 핵심 에이전트 기술 스택

| 구분 | 적용 기술 | 적용 목적 |
|---|---|---|
| LLM | Azure OpenAI | 문서 분석, 사용자 질문 이해 및 자연어 응답 생성 |
| Agent Workflow | LangGraph | AI Wiki 생성 및 질의응답 워크플로우 구성 |
| Embedding | text-embedding-3-small | Wiki 콘텐츠의 벡터 임베딩 생성 |
| Vector DB | Qdrant | 벡터 유사도 기반 Wiki 검색 및 RAG 컨텍스트 제공 |
| Metadata DB | PostgreSQL | Knowledge Space, 원본 문서 및 Wiki 메타데이터 관리 |
| Backend API | FastAPI | Wiki Builder 및 AI Assistant API 제공 |
| Storage | Local File System | 업로드된 원본 문서 저장 |
