# 테스트 케이스 — Knowledge Space 생성

[user-scenarios.md](./user-scenarios.md)의 7개 시나리오(주 흐름 + 대안 흐름)를 인수 테스트 케이스로 관리한다. 구현/수정이 있을 때마다 관련 케이스를 재실행하고 상태를 갱신한다.

- 상태 값: ✅ Pass · ❌ Fail · ⬜ Not Run · 🚧 Blocked
- 각 케이스는 `user-scenarios.md`의 해당 절차/대안 흐름과 1:1로 대응한다. 시나리오 문구가 바뀌면 이 문서도 함께 갱신한다.
